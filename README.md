# 9 CSS Grid Layouts Project

This project demonstrates 9 different webpage layouts using CSS Grid. It is built using HTML5 and CSS3 and focuses on mastering modern layout techniques.

---

## 🚀 Live Demo

- GitHub Repository: https://github.com/Grid-Layouts-Professional
- Wasmer Deployment : https://grid-layouts-professional.wasmer.app/ 

---

## 📌 Project Description

This project contains 9 unique layout designs created with CSS Grid. Each layout demonstrates different grid concepts such as:

- Grid Template Columns
- Grid Template Areas
- Auto-fit & Auto-fill
- Responsive Design
- Grid Auto Flow
- Sidebar Layouts
- Dashboard Layouts
- Card-based Layouts
- Magazine-style Layouts
